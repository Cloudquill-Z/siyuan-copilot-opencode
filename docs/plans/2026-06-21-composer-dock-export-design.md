# 对话框边框、侧栏恢复与导出署名修复设计

## 目标

修复暗色模式下问答/修订状态边框顶部圆角缺失、插件重启后侧栏内容不自动恢复，以及下载会话时未使用最新自定义用户名的问题。

## 设计

- 对话框保留现有结构和配色，使用不参与布局、不会裁剪弹出菜单的伪元素完整绘制状态边框。
- Dock 恢复逻辑把“面板尚未生成”视为等待状态，通过有上限的重试等待按钮和面板就绪；仅在当前插件生命周期有效且组件尚未挂载时初始化。
- 文件导出在点击时重新读取持久化设置，并以最新 `userName` 构造导出上下文；空名称仍使用本地化的“用户”回退值。

## 验证

- 增加轻量回归检查，覆盖边框绘制规则、Dock 等待判定和自定义用户名导出。
- 执行回归检查、生产构建，并更新 Graphify 知识图谱。
