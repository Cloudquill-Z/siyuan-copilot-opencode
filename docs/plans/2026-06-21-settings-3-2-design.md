# 设置中心 3.2 设计

## 目标

修复导出署名和并发设置的完整数据流，并将设置页整理为清晰、克制、符合 SiYuan 原生体验的设置中心。

## 设计方向

- 左侧导航按功能呈现清楚的选中状态，底部说明“修改自动保存”。
- 右侧统一使用页面标题、简短说明和分区内容，通用设置行与定制面板采用同一密度、边框和间距。
- 只使用 `--b3-*` 主题变量，兼容亮色、暗色和窄窗口。
- 所有导出入口在执行时读取最新设置；并发上限既控制新建任务，也在真正启动任务前二次校验。

## 反馈与错误处理

- 保存中、已保存和保存失败具有明确状态。
- 达到并发上限时阻止新增或启动，并显示当前上限及处理建议。
- 数字型下拉选项保持数字类型，重新打开设置页后仍正确选中。
